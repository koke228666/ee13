## ee13
_experimental13_ - тема для [OpenVK](https://github.com/openvk/openvk), которая использует ресурсы и вёрстку ВКонтакте за ~2013 год.



#### как это ваще работает
в файле _[ee13_vars.latte](tpl/ee13_vars.latte)_ лежат функции для упрощения генерации ответов

_[badbrowser.php](tpl/About/BB.latte)_ был изменён под получение AJAX запросов от скриптов. пока что там реализована лишь она функция - изменение статуса (даже потом вряд ли чёто поменяется).
